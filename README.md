### Welcome to my FastAPI learning repository!

This repository documents my journey from writing basic API endpoints to designing a fully modular, and production-ready backend architecture.

it serves as both learning portfolio and an **educational resource for aspiring software engineers**. The current application is a fully functional Student Data Managemnet System that strictly adheres to the priciples of Separation of Concerns (SoC).

**For Recruiters and Engineering Managers** this repository demonstrates my focus on clean code, modular architecture, strict data validation, and REST API best practices.

## About FastAPI
FastAPI is a modern, high-performance web framework for building APIs with Python based on standard Python type hints.
it is built on:  
**Starlette**  
**Pydantic**  

## Tech Stack
**Python3.10+**  
**FastAPI**    
**Pydantic**  
**SQLAlchemy**    
**Pytest**    

## Getting Started  
**Clone the repository**  
**Create a virtual environment(Recommeded)**  
**Install dependencies**  
**Run the application**  


### Interactive API Documentation
FastAPI automatically generates interactive Swagger UI documentation. Once the server is running navigate to -  

**Swagger UI: http://127.0.0.1:8000/docs**  
**ReDoc: http://127.0.0.1:8000/redoc**
